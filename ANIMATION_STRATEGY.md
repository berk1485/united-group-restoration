## ANIMATION OPPORTUNITIES FOR ROOFING/RESTORATION

### **Niche 1: "Before/After Storytelling" (Emotional Impact)**

**Concept:** As user scrolls, show actual damage progression → restoration journey

**Animations:**
```
Section 1 (Hero):
- Parallax background (storm clouds moving)
- Title slides in from left
- CTA button pulses (trust signal)

Section 2 (Problem):
- Before image zooms in (damage close-up)
- Text slides in from right with damage description
- Icon animations (cracked roof, water damage)

Section 3 (Solution):
- After image slides in from left
- Before image fades behind
- Workers animated (construction elements)
- Text reveals: "Within 48 hours..."

Section 4 (Gallery):
- Images fade in as scrolling
- Hover: Lightbox expand animation
- Project counts animate up (23 homes restored, etc.)

Section 5 (Testimonials):
- Cards flip in on scroll
- Star ratings animate fill
- Customer photos fade from blur to sharp
```

**Tools:** GSAP ScrollTrigger + custom CSS animations

**Niche advantage:** Emotional connection (not just "we fix roofs" but "we save your home")

## TECHNICAL IMPLEMENTATION

### **Framework Choice**

**Option A: GSAP ScrollTrigger (Recommended for your case)**
- Pros: Smooth, performant, great for scroll-linked animations, works with vanilla JS
- Cons: Requires learning GSAP syntax
- Cost: Free (with paid pro plugins)
- Best for: Scroll animations, parallax, complex timelines

**Option B: Framer Motion**
- Pros: React-native, declarative, great for component animations
- Cons: Requires React refactor (your site is vanilla + Tailwind)
- Cost: Free
- Best for: Component-level animations, page transitions

**Option C: Custom CSS + Intersection Observer**
- Pros: Lightweight, no dependencies
- Cons: Harder to coordinate complex sequences
- Cost: Free
- Best for: Simple fade-in/slide animations

**Recommendation: GSAP ScrollTrigger** (perfect balance of power + ease for your use case)
